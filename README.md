# FFmpegPlayer
基于FFmpeg的Android视频播放器

#### 编译环境
FFmpeg版本： **4.2.2**    
NDK版本：**r17c** 
#### 运行环境
* x86(模拟器)
* arm64-v8a(64位手机)
#### 功能点
* 支持播放视频（音视频已同步）
* rtmp直播流(比如：湖南卫视，广西卫视)
